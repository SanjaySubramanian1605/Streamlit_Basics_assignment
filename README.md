# 📊 Sales Summary Dashboard

An interactive **Sales Dashboard** built using **Streamlit** and **Pandas**.  
This application allows users to filter sales data by category and visualize it using charts in a simple and user-friendly interface.

---

## 🚀 Features

- 📌 View sales data in a table format  
- 🔍 Filter products by category  
- 📈 Interactive sales visualization using line charts  
- ⚡ Fast and lightweight Streamlit application  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit  
- Pandas
